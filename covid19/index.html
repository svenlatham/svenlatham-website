About these pages

Covid-19 has had widespread effects on society and economics. This is a small collection of resources looking at the effects of the pandemic, likely eventual recovery plans (UK-based perspectives) and tracking updates.

These pages will be updated frequently. Please check back for more!

This is a personal project run by Sven Latham. I have interests in data in town centres, so these pages will have a particular bias.

Geeky bit: This is all on Github if you want to see the source. I'm using it as an excuse to learn Jekyll as well ;-)
